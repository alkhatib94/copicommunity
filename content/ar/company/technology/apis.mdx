# واجهات البرمجة (APIs)  

**API - واجهة برمجة التطبيقات**  

تم بناء الـ **APIs** على إطار عمل **Microsoft .Net Framework**، وتُستخدم لتمكين تطبيقاتنا من التواصل مع بعضها البعض.  

توفر الـ APIs وسيلة قياسية للمطورين للتواصل مع قاعدة البيانات الخلفية، لإرسال واستقبال معلومات اللاعبين، وتتبع النقاط والمستويات الخاصة بالمناطق (Districts) وغيرها.  

</br>  

## أهم الـ APIs  

<ul className="styled-list">
  <li><b>Play-and-Earn API (P+E):</b> يتم تحديث المكافآت المكتسبة عبر واجهة P2E API.</li>
  <li><b>Build-and-Earn و NFT API (B+E):</b> عند قيام اللاعبين بإنشاء الـ NFTs الخاصة بهم، يتم إرسال المعلومات إلى "المقر" عبر واجهتي الـ NFT API و B+E API.</li>
  <li><b>Player Profile API:</b> تُستخدم هذه الواجهة لاسترجاع وتحديث بيانات شخصية اللاعب الحالي.</li>
  <li><b>Learn-and-Earn API (L+E):</b> يتم نقل المعلومات الخاصة بالتعلم داخل Cornucopias عبر واجهة L+E API.</li>
  <li><b>Host-and-Earn API (H+E):</b> يتم تحديث المكافآت المكتسبة من خلال استضافة العقد عبر واجهة H+E API.</li>
  <li><b>Blockchain API:</b> تُنقل معلومات محافظ اللاعبين على البلوكشين عبر واجهتي NFT API و Blockchain API.</li>
</ul>
