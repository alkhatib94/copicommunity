# Wood

Wood is an essential crafting resource in Cornucopias. This resource can be used to craft tools, buildings and technology within Cornucopias.

<div className="note-box success">
  <strong>Note:</strong> The life cycle begins with trees in forests which are cut using axes and converted into logs. These logs are then taken to a sawmill and converted into wood beams by using a table saw. Wood beams can then be taken to a lathe to be converted into specific crafting elements.
</div>

## Trees

Trees are environment features found scattered across themed zones. When chopped using an axe, trees are stored in the inventory as logs.  

Some trees can only be found in certain zones and can be level restricted. Different types of trees will have different aesthetic characteristics.

## Wood Beams

Wood beams are versatile crafting materials which are sourced from various types of trees and are essential for tools and building. They are acquired by processing logs at a sawmill.
