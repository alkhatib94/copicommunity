# Calido Valley Pavilion

The **Calido Valley Pavilion** can be found in the **Calido Valley Mega Dome**.

This is an area you get to via the **Calido Valley Resort lifts** near the **Grand Valley Hotel**.  
It is a **shopping mall** that has a number of attractions including the mini game **Rumble Ball**.  
It is also the place you go if you want to race in the fast-paced **Cornucopias Racing League**.
