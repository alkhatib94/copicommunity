# Public Testing  

**CURRENT PHASE: Pre Alpha Testing**  

Public Testing was opened on **2nd August 2023**. Over the following months more people were invited to test and more servers were opened up throughout the World.  

Alongside the Cornucopias Game there is a PC Application known as the  
[**Game Launcher**](/game/launcher), this detects the device hardware and links to the players Web3 wallet to control game access while in the [Testing Phase](/company/technology/game-devices/pc-desktop/public-testing).
