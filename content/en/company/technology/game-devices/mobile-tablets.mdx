# Mobile Tablet Devices  

The mobile tablet version of Cornucopias will not be the same as the PC version.  

Cornucopias will release various types of applications - from **Play-and-Earn mini-games** to **management apps** (e.g., Land and Guild Management).  

Cornucopias will be available on mobile tablets from the following manufacturers:  

<ul className="styled-list">
  <li>Google</li>
  <li>Samsung</li>
  <li>Apple</li>
</ul>  

For the latest release schedule, see the <a href="/game/about-the-game/road-map" className="underline">Roadmap</a>.
