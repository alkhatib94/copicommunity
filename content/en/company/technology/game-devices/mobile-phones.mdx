# Mobile Phone Devices  

Cornucopias plans to release various types of applications - from **Play-and-Earn** mini-games to mobile **management apps** (e.g., Land and Guild Management).  

Cornucopias will be available on mobile devices that use the following operating systems:  

<ul className="styled-list">
  <li>Google Android</li>
  <li>Apple iOS</li>
</ul>

For the latest release schedule, see the <a href="/game/about-the-game/road-map" className="underline">Roadmap</a>.
