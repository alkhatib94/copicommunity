<div className="note-box danger transparent">
  If you plan on setting up your node pool locally you might want to check to make sure your network supports port forwarding (required for nodes) natively.
</div>

<br />

On the local machine you plan to run the node pool on, open a command prompt (cmd on Windows) and type:

<div
  className="rounded-xl border border-white/10 bg-zinc-900/60 p-4 font-mono text-sm overflow-x-auto"
  dir="ltr"
>
  <pre className="whitespace-pre-wrap m-0"><code className="language-bash">tracert [your_ip_address]</code></pre>
</div>

Replace <code>[your_ip_address]</code> with your actual IP address (you can search “what is my ip” online).  
You should only see <b>one hop/line</b>. If you see multiple, your network does not support port forwarding natively.
