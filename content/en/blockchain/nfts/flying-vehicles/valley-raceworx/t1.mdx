# Valley Raceworx T1

<div className="note-box ">
  <strong>Not an NFT:</strong> The Valley Raceworx <b>T1</b> is <u>not</u> an NFT - all models will be available to use in-game for everyone.
</div>

The <b>Valley Raceworx T1</b> is a flying vehicle built purely for racing and can be raced in <b>Calido Valley</b>.  
It comes in <b>3 models</b> that map to <b>3 race classes</b> - A, B and C - with careful balance across the class system.

<figure className="img-center">
  <img src="/images/nfts/vrx/t1/vrx-t1-hero.avif" alt="Valley Raceworx T1 - Hero render" />
  <figcaption>Valley Raceworx T1 - built for pure racing.</figcaption>
</figure>

A deep dive into the 3 different T1s can be seen in <b>COPI Cafe Episode 58</b>.

</br>

## Difference in Class

### Class C
Class C is the entry level and the slowest of the T1s. It has the best handling of all 3.

<figure className="img-center">
  <img src="/images/nfts/vrx/t1/class-c.avif" alt="VRX T1 - Class C" />
  <figcaption>VRX T1 - Class C focus on control & handling.</figcaption>
</figure>

### Class B
The T1 - Class B has <b>reverse thrusters</b> (making braking fun and exhilarating), a faster top speed than Class C, and an <b>Aero Braking kit</b>.

<figure className="img-center">
  <img src="/images/nfts/vrx/t1/class-b.avif" alt="VRX T1 - Class B" />
  <figcaption>VRX T1 - Class B with reverse thrusters & aero braking.</figcaption>
</figure>

### Class A
The T1 - Class A also has <b>reverse thrusters</b>, and is faster than both Class B and Class C.

<figure className="img-center">
  <img src="/images/nfts/vrx/t1/class-a.avif" alt="VRX T1 - Class A" />
  <figcaption>VRX T1 - Class A maximum performance.</figcaption>
</figure>
